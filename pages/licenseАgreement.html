3. Политика конфиденциальности

3.1. Собираемая и отправляемая информация. 3.1.1.При установке приложение создает уникальный идентификатор в виде буквенно-числового кода (далее UID) на основе информации о вашей копии операционной системы Windows. UID строится таким образом, чтобы информация, на основе который он был создан, никак не могла быть восстановлена. UID отправляется на наш сервер в момент запроса наличия обновлений для данного ПО, и используется для определения количества активных пользователей (популярности приложения). Помимо этого, на сервер обновлений отправляется информация об используемой локализации. Эта информация необходима для того, чтобы определить на каком языке сервер должен вернуть список изменений. Вы можете отказаться от передачи это информации, путем отключения плагина, отвечающего за проверку обновлений, однако, в этом случае вы перестанете получать обновления в полуавтоматическом режиме. 3.2. Пользовательские данные и их хранение.

3.2.1. Вся информация, полученная от сторонних сервисов, на которых авторизовался пользователь через данное ПО, используется лишь для отображения непосредственно пользователю и НЕ передаются на наши сервера или сервера наших партнеров, за исключением серверов самих сервисов. Данное ПО может хранить локальную копию полученных данных, с целью уменьшения количества запросов к сервисам. Данные хранятся исключительно в папке с настройками программы в бинарном (в нечитаемом без специальных средств) виде. Локальные копии данных удаляются автоматически при отзыве авторизации (см. п.3.5) или удалении данного ПО целиком.
